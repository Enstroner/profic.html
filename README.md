# PROFiC DASH

A fun Geometry Dash style rhythm runner made in a single HTML file.

![Game Screenshot](screenshot.png)

## How to Play
- Press **Space** or **Click/Tap** to jump
- Hold to fly in ship mode
- Avoid the obstacles and beat your high score!
